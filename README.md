# **Fate Naruto Ref (Fate Fight Game)**
##🎮 Play the Game Here! https://gx.games/games/6fer6m/fate-weirdo-type/

Welcome to Fate Naruto Ref, an action-packed 2D fighting/platformer game made in GameMaker Studio! This project is heavily inspired by and built upon the original Naruto Fight Game framework by deratsonder. It takes the core ninja combat mechanics and modifies them to create a unique, "Fate"-inspired crossover experience.

##🌟 Features
Dynamic Combat System: Engage in fast-paced battles using iconic abilities like Rasengan, Shinra Tensei, and Kaiton (Fire Style).

Epic Bosses & Enemies: Fight your way through challenging foes including Pain, Obito, and Zetsu.

Massive Summons: Face off against or utilize ultimate powers like Kurama (Nine-Tails) and the Susanoo.

Multiple Stages: Battle across multiple rooms and environments (Mountains, Trapped areas, and more), ending in epic climax battles.

Health & Danger Mechanics: Carefully manage your HP, avoid traps, and utilize healing items (O_HoiHP) to survive the gauntlet.

##🕹️ How to Play
(Note: Adjust these controls if you have specific key bindings in your GameMaker project)

Movement: Arrow Keys / WASD

Attack / Basic Jutsu: Z / Left Mouse Click

Special Abilities: X, C / Number Keys

Restart: Press R to restart the room/level

Play Again: Press S on the game over screen

##🛠️ Development & Building
This game was built using GameMaker Studio 2.

To run or modify the project locally:

Clone or download this repository.

Open GameMaker Studio.

Click on Open Project and select the Naruto Fight.yyp file.

Press F5 (or the Play button) to compile and run the game.
